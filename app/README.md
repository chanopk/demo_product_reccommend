# Financial Product Recommendation Demo

แอปพลิเคชันรูปแบบใหม่สำหรับนำเสนอผลิตภัณฑ์ทางการเงินด้วย AI สำหรับผู้จัดการความสัมพันธ์ลูกค้า (RM)

![Demo Screenshot](./public/images/banking-hero.jpg)

## ฟีเจอร์หลัก (Key Features)

แอปพลิเคชันนี้มอบชุดเครื่องมือ AI อัจฉริยะเพื่อช่วยสนับสนุนการทำงานของ RM (Relationship Manager) ให้มีประสิทธิภาพสูงสุด:

### 1. แดชบอร์ดข้อมูลลูกค้า (Customer Dashboard)
หน้าจอสรุปข้อมูลลูกค้าแบบครบวงจร 360 องศา ประกอบด้วย:
- **ข้อมูลส่วนตัว**: ชื่อ, อายุ, อาชีพ และ Segment ของลูกค้า
- **สถานะทางการเงิน**: มูลค่าพอร์ตการลงทุนปัจจุบัน, ระยะเวลาที่เป็นลูกค้า (Loyalty), และผลิตภัณฑ์ที่มีอยู่
- **ระดับความเสี่ยง**: คะแนนความเสี่ยง (Risk Score 1-10) พร้อมตัวชี้วัดความเปราะบาง (Vulnerability) เพื่อช่วยให้แนะนำผลิตภัณฑ์ได้เหมาะสมที่สุด

### 2. ระบบแนะนำผลิตภัณฑ์ (Product Recommendation)
ระบบอัจฉริยะที่ผสานพลัง AI และ Data Analytics เพื่อคัดสรรผลิตภัณฑ์ที่ใช่:
- **Personalized Matching**: แนะนำกองทุนและผลิตภัณฑ์ที่ตรงกับไลฟ์สไตล์และเป้าหมายของลูกค้าแต่ละราย
- **การเปรียบเทียบข้อมูล**: ตารางเปรียบเทียบผลตอบแทน, ระดับความเสี่ยง และระยะเวลาการลงทุนให้เห็นภาพชัดเจน
- **การวางแผนตามเป้าหมาย**: จำลองผลตอบแทนสำหรับเป้าหมายสำคัญ เช่น วางแผนเกษียณ, สร้างความมั่งคั่ง หรือทุนการศึกษาบุตร

### 3. บริการที่ปรึกษาความเสี่ยง (Risk Consultant Service)
บริการจำลองสถานการณ์ความเสี่ยงด้วย AI (Stress Testing):
- **Portfolio Stress Testing**: ทดสอบพอร์ตการลงทุนในสภาวะตลาดวิกฤติต่างๆ เพื่อดูผลกระทบที่อาจเกิดขึ้น
- **Visual Impact Analysis**: แสดงภาพความเสี่ยงให้ RM และลูกค้าเข้าใจได้ง่าย
- **Sales Enablement**: ข้อมูลสนับสนุนการตัดสินใจเพื่อเสนอผลิตภัณฑ์ป้องกันความเสี่ยง (Hedging) หรือกระจายความเสี่ยง (Diversification)

### 4. แชทบอทอัจฉริยะ (Smart Chat Agent)
ผู้ช่วย AI ส่วนตัวที่พร้อมตอบทุกคำถาม:
- **ข้อมูลผลิตภัณฑ์ครบถ้วน**: สามารถสอบถามรายละเอียด เงื่อนไข และค่าธรรมเนียมของผลิตภัณฑ์ทุกตัวได้ทันที
- **สนับสนุนการขาย**: ช่วยตอบข้อโต้แย้งลูกค้า หรือค้นหาจุดเด่นของผลิตภัณฑ์เพื่อปิดการขายได้รวดเร็วขึ้น
- **Operational Support**: ช่วยค้นหากระบวนการทำงานหรือเอกสารที่จำเป็นต้องใช้

### 5. เครื่องมือช่วยเหลือ (Essential Tools)
- **ค้นหาสาขา (Branch Locator)**: แผนที่อินเทอร์แอกทีฟพร้อมระบบนำทางไปยังสาขาที่ใกล้ที่สุด
- **บริการรับฝากสินทรัพย์ (Custodian Services)**: ช่องทางอำนวยความสะดวกสำหรับการเปิดบัญชี Custodian

## อนาคตและฟีเจอร์ที่น่าสนใจ (Future Roadmap)
แนวทางการพัฒนาฟีเจอร์ AI เพิ่มเติมเพื่อยกระดับแพลตฟอร์ม:

### 1. ตรวจสุขภาพทางการเงิน (Financial Health Check & Optimization)
ระบบ AI วินิจฉัยสุขภาพการเงิน โดยวิเคราะห์รายรับ-รายจ่ายและหนี้สิน เพื่อประเมิน "คะแนนสุขภาพการเงิน" พร้อมแนะนำวิธีปรับปรุง เช่น การรวมหนี้ หรือการวางแผนภาษี

### 2. แผนนำทางความมั่งคั่งแบบพลวัต (Goal-Based Wealth Navigation)
ระบบวางแผนการเงินที่ปรับเปลี่ยนได้ตามสถานการณ์จริง ผู้ใช้กำหนดเป้าหมาย (เช่น ซื้อบ้าน, เกษียณเร็ว) และ AI จะคอยปรับเปลี่ยนแผนการลงทุนให้อัตโนมัติเมื่อตลาดเปลี่ยนแปลง หรือมีเหตุการณ์สำคัญในชีวิต

### 3. ระบบจับตาข่าวสารและอารมณ์ตลาด (Sentiment & Market Intelligence)
ระบบวิเคราะห์ขาวสารและ Social Sentiment แบบเรียลไทม์ที่เกี่ยวข้องกับพอร์ตลูกค้า พร้อมแจ้งเตือนทันทีเมื่อมีเหตุการณ์สำคัญ (เช่น "ตรวจพบความผันผวนในกลุ่มเทคโนโลยี - ควรทบทวนพอร์ตการลงทุน")

### 4. ระบบแนะนำสิ่งที่ควรทำถัดไป (Next Best Action - NBA)
ระบบวิเคราะห์ทำนายพฤติกรรมเพื่อแนะนำ Action ที่ดีที่สุดสำหรับ RM ในขณะนั้น ไม่ว่าจะเป็นการเสนอสิทธิพิเศษไลฟ์สไตล์, การอวยพรวันเกิด หรือจังหวะการปรับพอร์ตที่เหมาะสมที่สุดในช่วงเวลานั้น

## เริ่มต้นใช้งาน (Getting Started)

### สิ่งที่ต้องมี (Prerequisites)
- Node.js 20+
- npm หรือ yarn

### การติดตั้ง (Installation)

1. Clone ทรัพยากร (Repository):
```bash
git clone <repository-url>
cd app
```

2. ติดตั้ง Dependencies:
```bash
npm install
```

3. เริ่มต้นเซิร์ฟเวอร์สำหรับพัฒนา:
```bash
npm run dev
```

4. เปิดเบราว์เซอร์ไปที่ `http://localhost:5173`

### Build สำหรับ Production

```bash
npm run build
```

ไฟล์ที่ Build แล้วจะอยู่ที่โฟลเดอร์ `dist/`
